# 咖啡荒谬人格研究所 V2

这是一个借鉴“娱乐化人格测试传播机制”的咖啡人格网站。它不复制任何具体测试题或视觉资产，而是把有梗结果、隐藏人格、可分享海报与真实咖啡消费画像结合起来。

## 直接运行

解压后双击 `index.html` 即可。

为避免部分浏览器限制本地文件的分享和剪贴板功能，也可以在目录内启动一个简单静态服务器：

```bash
python -m http.server 8080
```

然后访问 `http://localhost:8080`。

## V2 功能

- 24 道生活情境题
- 六维人格评分：新奇、仪式、精研、社交、效率、审美
- 12 种常规人格
- 4 种隐藏人格
- 主人格 + 次人格
- 六维雷达图
- 咖啡依赖值、抽象浓度、社交扩散力
- 自动生成 1080 × 1440 PNG 结果海报
- 原生系统分享；不支持时自动复制文案
- 手机端响应式布局
- 测试进度与最近结果保存在浏览器本地
- 无后端、无数据库、无框架依赖

## 商业化下一步

当前版本所有数据只保存在用户本地。要用于真实社群运营，建议增加：

1. 用户授权与匿名测试记录 API
2. 人格结果对应的咖啡豆、门店和活动推荐
3. 分享邀请码与好友人格关系图
4. 社群分组和城市活动匹配
5. 管理后台：完成率、题目流失率、人格分布、转化率
6. A/B 测试：题目、人格名称、结果页和推荐商品
